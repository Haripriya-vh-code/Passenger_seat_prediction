## Dataset

The datasets used in this project were provided as part of the **RedBus Data Hackathon**. Since they exceed GitHub's file size limits, they are hosted externally and can be downloaded using the links below.

### **1. `train.csv`**

Contains route and journey information used for model training.

**Download:**  
https://drive.google.com/file/d/1m62y2NynX3WRx3zJDF8VrzNWBdAN4D9E/view?usp=sharing

---

### **2. `transactions.csv`**

Contains historical booking and transaction information for each route.

**Download:**  
https://drive.google.com/file/d/1LVxW_KEH3QWBHALQjGq2EKfBeKp2F4zk/view?usp=sharing
